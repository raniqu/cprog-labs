## 1. Условие задания

- По запросу с клавиатуры в консольном приложении вводится строка, которая может содержать:
    - знаки операций: +, -, *, /
    - константы (целые или вещественные)
    - скобки
    - имена функций: sin, cos, tg, ctg, exp
    - переменная x

- Реализована проверка правильности введённого выражения
- В случае ошибок ввода выдаются понятные сообщения об ошибках
- Если в формуле присутствует переменная x, программа запрашивает её значение, затем вычисляется результат с подстановкой значения x



## 2. Схема алгоритма
1. Ввод выражения 
2. Удаление пробелов
3. Разбивка на токены
   - Числа (только положительные)
   - Переменная "x"
   - Функции
   - Скобки
   - Операторы
   - Унарный минус -> "u-"
4. Проверка на наличие "x" (если есть, то запрашиваем х)
5. Алгоритм Дейкстры
   - приоритеты: (3) унарный минус, (2) умножение/деление, (1) сложение/вычитание
   - функции - высокий приоритет

   
6. Вычисление 
   - Число/x → push в стек
   - "u-" → pop 1 значение, push(-значение)
   - Бинарные операторы (+, -, *, /) → pop 2 значения, push(результат)
   - Функции → pop 1 значение, push(результат функции)

7. Результат (или выводим ошибку)

## 3. Контрольный расчет
На вход подается (sin x + 1) / (-2 * (5 + x))
